View and filter model findings

Published

December 4, 2024

Access a complete list of filterable findings logged across all your models in the ValidMind Platform, or look at only findings linked to a specific model.

Prerequisites

View model findings

You can review findings for all models at once, or only look at findings linked to a specific model.

All model findings

To view findings for all models:

  1. In the left sidebar, click Model Findings.

    In the page that opens is a list of all model findings, including information for:

    • Title
    • Severity
    • Status
    • Inventory model
    • Risk area
    • Assignee
    • Due date
    • Creation date
Changes are automatically saved to your account and do not affect other users.

Filter this list to include only findings you want to see,4 or toggle visibilty for column headers that can be dragged and dropped to change the order with the Manage Columns menu.

  1. Click on any model finding for more information, including the proposed remediation plan and any supporting attachments.5

    Attachments are listed with the following information:

    • Filename
    • The user who uploaded the file
    • File size

    Click on the attachment to view or download the file.

Model-specific findings

To view findings linked to a specific model:

  1. In the left sidebar, click .

  2. Select a model or find your model by applying a filter or searching for it.6

  3. In the left sidebar that appears for your model, click Model Findings.

    In the page that opens is a list of findings associated with that particular model, including information for:

    • Title
    • Description
    • Severity by color
    • Status
    • Assignee
    • Due date

  4. Click on any model finding for more information, including the proposed remediation plan and any supporting attachments.7

    Attachments are listed with the following information:

    • Filename
    • The user who uploaded the file
    • File size

    Click on the attachment to view or download the file.

Filter model findings

By default, model findings for all models are listed by creation date in descending order.

To narrow down the findings, you can apply some filters:

  1. In the left sidebar, click Model Findings for the list of all model findings.

  2. Click Filter.

  3. On the Select Your Filters dialog that opens, enter your filtering criteria for:

    • Status
    • Model inventory
    • Risk area
    • Business units
    • Severity
    • Assignee

    For example, to filter by open findings for the Customer Churn Model assigned to Jane Doe:

  4. Click Apply Filters.

Filters can be removed by clicking on the next to them on the main Model Findings page where the results are displayed.
